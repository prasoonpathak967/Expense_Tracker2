# 💰 Expense Tracker API (Spring Boot)

A RESTful Expense Management Application built using **Spring Boot**, **Spring Data JPA**, and **Maven**.

This project allows users to perform CRUD operations (Create, Read, Update, Delete) on expenses and demonstrates a clean layered architecture following industry best practices.

---

## 🚀 Project Overview

The Expense Tracker API is a backend application that helps manage daily expenses efficiently.  
It provides REST endpoints to add, retrieve, update, and delete expense records.

This project demonstrates:

- REST API development
- Layered Architecture (Controller, Service, Repository)
- Spring Boot configuration
- Maven project structure
- Database integration using JPA

---

## 🛠️ Tech Stack

- Java 17+
- Spring Boot
- Spring Data JPA
- Maven
- H2 / MySQL Database
- REST API
- Git & GitHub
